This project was created by 3 Loyola students in a course focusing on open source software. The inspiration for the game came from an earlier project one of the students had created in Python and we knew that we could find a way to make it bigger and better!

Our mission for this project has been to create a game that people not only enjoy playing, but one that helps foster collaboration and contribution between players. We believe this is what really makes a great game.

If you would like to join our team, please see our contribution page for details on how you can help!
