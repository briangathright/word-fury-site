What are you using to develop this game?
We are using HTML5 and phaser.
I have never heard of phaser, what is it?
Phaser is an open source HTML5 framework for building games. We originally played around with different technologies, but one of our goals was to have this hosted on the website for users to play. Having the game built around HTML made this possible. Definitely checkout the Phaser site to contribute or make your own game!
Where is the repository so I can contribute?
The code repository is hosted on our repo on GitHub. The best way to contribute is to fork the project, add in a new functionality or bug fix, and then create a new pull request so that the community can review. It's also a good idea to join our Google group where we do our communication.
Is this an open source project?
Yes, it is being released under a General Public License (GPL).
Where are your developer guidelines and game instructions located?
Both of these are located on the README file in the GitHub Repo.
Where is the download area?
You can download the source code on our Github Repo.
Where is the user manual?
The game is very simple to play. Simply type the falling words before they hit the bottom. For more in depth instructions, checkout the README file for more instructions.
What is the address of your mailing list?
We communicate mainly through our Google Group. We found this gives the most visibility to conversations and allows people to opt out of the group when they please.
What issue tracker will you use?
We use GitHub's built in issue tracker. We found it beneficial to have the code repository located in the same area as our issue tracker. You can find the issue tracker here.
Are you planning on expanding the game functionality?
Absolutely! We have a tons of ideas for the game that we would like to add. And just so we are clear, we are always open to help and new ideas!
