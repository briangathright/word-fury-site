Related projects:

Falling Words Game : typingstudy

http://www.typingstudy.com/games/falling_words
This game features falling words and a scoring system as well as a lives system. The higher levels feature harder words + the previous words and words spawn faster. Multiple words can fall at once.


Falling Words : freeonlinetypinggames

http://freeonlinetypinggames.com/free-online-typing-games/falling-words-game
This game is very similar to the one above. Except it has better graphics and minimal sound effects. It has no scoring system. It still uses lives and for each life lost it becomes easier to lose another as the death zone increases in size. It also has no level up system or difficulty setting. Our game has a different scoring system.


Falling Words : powertyping

http://www.powertyping.com/trees.shtml
This one has no sound and minimal graphics. It does have a high scores feature. It has word categories and difficulty settings (all speed based). You lose in a similar way to the one above where each missed word increases the danger zone.

Our scoring system is based on speed whereas this game scoring is based on words length mostly. Additionally, our leveling system will feature harder words as well as other difficulty enhancers such as faster falling speeds, color changing words, moving words (left-right, spinning etc.) as well as obstacles that obscure words. Additionally, we will have a high scores system that tracks the best players. We will also have sound effects / music and more advanced art. As well as other features such as power ups, jumbled words, etc.


WordRage : mindjolt

http://www.mindjolt.com/wordrage.html
We originally planned on naming our game Word-Rage; however we found a word game with the same name already existed. Therefore we changed the name of our project to Word Fury. The game itself is in a different category (it is a word building/puzzle game)


Typing Defense : wordgames

http://www.wordgames.com/typing-defense.html
Level up system, high scores, defend the middle of the screen, lots of graphics, music, sound effects
powerups : slow; freeze; kill all
words break into smaller words (stems broken -> broke)
combos, accuracy rating, smart type (picks up any word that has that stem)
